## About
Website built to provide the Vesa Community ([Discord server](https://discord.gg/vesaleague)) a way to be involved in other divisional matches. Each night players can draft a team and players to see who scores the best!

Uses Discord as an OAuth sign in.

Uses PostgreSQL for db management.

Hosted on Vercel

## Getting Started

The only required environment variables are:
- AUTH_DISCORD_ID (acquired from [Discord](https://discord.com/developers/applications))
- AUTH_DISCORD_SECRET (acquired from [Discord](https://discord.com/developers/applications))
- AUTH_SECRET (generated by `npx auth`)
- DATABASE_URL (provided by preferred database, I am using Neon)


Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
